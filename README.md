## Getting started

```bash
# clone the project
git clone https://github.com/NelsonEAX/vue-vuetify-admin.git

# enter the project directory
cd vue-vuetify-admin

# install dependency
npm install

# develop
npm run serve
```

This will automatically open http://localhost:8080

## Build and Test

```bash
# build
npm run build

# run e2e test
npm run test:e2e

# run unit test
npm run test:unit
```
